Installation¶

Installing Audio convert is easy but you have to do it from the command line.

  1. Goto the github page: https://github.com/cavpp/audio_convert/tree/master/dist and download the latest version.

  2. Open terminal

  3. Type “pip install”, followed by the gz file.

    For example:

    pip install CAVPP_Audio_Convert-0.1.1.tar.gz

  4. You are done. The audio conversion script is installed.
