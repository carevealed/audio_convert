Usage¶

Command Line¶

In a terminal windows, simple type “audioConvert.py” followed by a single file or directory.

Example 1: Single wav file:

audioconvert.py /Users/lpsdesk/PycharmProjects/audioConvert/OtherFolder/ALLEY_B.wav

Example 2: Entire folder of wav files:

audioconvert.py /Users/lpsdesk/PycharmProjects/audioConvert/testFolder

Graphical User Interface¶

In a terminal windows, simple type the following command:

audioConvert.py -g
